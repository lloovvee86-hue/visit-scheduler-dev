# 🚀 [Free!] 0원 과금 방문 일정 수립 서비스 (Visit Scheduler)

AI 1인 기업 대표님과 영업 전사들을 위한 **최강의 동선 최적화 서비스**입니다! 🫡 🐟
네이버 API 유료화에 당황하셨나요? 카카오 API를 활용해 **완전 무료**로 스마트한 일정을 관리하세요.

---

## ✨ 주요 기능

- **지도 전환 (Kakao Maps):** 일 30만 건 무료 쿼터를 제공하는 카카오 지도로 무제한 사용 가능!
- **세부 일정표:** 출발 시간 설정부터 장소별 도착/출발 시간을 초단위로 자동 계산.
- **체류 시간 관리:** 각 미팅 장소에서의 소요 시간을 입력하면 전체 일정이 실시간 업데이트.
- **비즈니스 메모 (통합 카드):** 장소별 미팅 안건, 주차 정보 등을 기록하고 주행 정보와 함께 한눈에 확인.
- **로드 브리핑:** "국도 >> 고속도로" 시퀀스 뷰와 IC/JC 상세 안내로 운전 중 가독성 극대화.
- **동선 최적화 (2-Opt):** 이동 거리를 최소화하는 알고리즘으로 기름값과 시간을 아껴드립니다.

---

## 🛠️ 기술 스택

- **Frontend:** Vanilla JS, HTML5, CSS3 (Premium Dark Mode)
- **Backend:** Python Flask (Kakao API Proxy)
- **Maps API:** Kakao Maps SDK, Kakao Mobility Directions REST API

---

## 🚀 시작하기

### 1. 환경 설정
`app.js` 또는 로컬 스토리지에 카카오 API 키를 입력합니다.
- JavaScript 키 (Frontend용)
- REST API 키 (Backend용)

### 2. 백엔드 서버 실행
보안 및 CORS 이슈 해결을 위해 프록시 서버를 실행합니다.
```bash
pip install flask flask-cors requests
python server.py
```

### 3. 프론트엔드 접속
`index.html` 파일을 브라우저로 열거나, 라이브 서버로 실행하세요.
